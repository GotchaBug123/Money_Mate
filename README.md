# Smart_RoboAdviser

# 🚀 로보어드바이저: 마이데이터 기반 지출 분석 및 목표 달성형 자산 관리

> **"당신의 소비 속에 숨겨진 투자금을 찾아드립니다."**
> 마이데이터와 AI를 결합하여 사회초년생의 돈 관리 습관을 혁신하는 목표 달성형 금융 비서

---

## 🧐 Why? (기획 배경)
초보 투자자, 특히 대학생과 사회초년생은 항상 다음과 같은 벽에 부딪힙니다.
1. **불투명한 투자 여력:** "이번 달 생활비 쓰고 남은 돈, 무작정 투자해도 안전할까?"
2. **해석 불가능한 금융 정보:** DART 공시, 재무제표... 초보자에게는 읽기 힘든 외계어입니다.
3. **감정적인 투자 결정:** 데이터가 아닌 공포와 욕심에 기반한 비합리적인 선택을 반복합니다.

우리는 이 문제를 객관적 데이터(마이데이터)와 AI(LLM)로 해결하여, 누구나 합리적인 투자 습관을 가질 수 있도록 돕습니다.

---

## ✨ Key Features (핵심 기능)

### 📊 1. 마이데이터 기반 지출 분석 (Spend Analysis)
* 단순히 지출 내역을 나열하는 것이 아니라, 소비 패턴을 정밀 분석합니다.
* AI가 계산한 '진짜 투자 가능한 금액'을 산출하여 현실 기반의 투자 가이드를 제공합니다.

### 🤖 2. AI 투자 통역사 (LLM-based Analysis)
* LLM(거대언어모델)이 복잡한 기업 공시(DART)와 투자 지표를 실시간으로 분석합니다.
* 전문가의 언어를 "이 회사는 빚이 적어 위기에 강해요"와 같이 누구나 이해할 수 있는 쉬운 언어로 번역해 드립니다.

### 🎲 3. 수학적 목표 시뮬레이션 (Monte Carlo Prediction)
* 금융공학 기법인 몬테카를로 시뮬레이션을 통해 "내 목표가 $n$년 뒤에 이루어질 확률"을 수치로 보여줍니다.
* 막연한 희망이 아닌, 수학적 근거에 기반한 자산 성장 시나리오를 제시합니다.

### 📉 4. 행동 변화 유도: 절약 챌린지 (Nudge)
* "커피 한 잔만 줄이면 투자 성공 확률이 $10\%$ 올라가요!"
* 지출 감소가 투자 수익으로 연결되는 과정을 시각화하여 사용자의 실질적인 행동 변화를 이끌어냅니다.

## 🛠 Tech Stack (기술 스택)

| 분류 | 기술 스택 | 역할 |
| :--- | :--- | :--- |
| **Frontend** | `React`, `Next.js` | 사용자 친화적인 인터페이스 및 대시보드 |
| **Backend** | `Spring Boot`, `JPA` | 금융 데이터 처리 및 시스템 안정성 확보 |
| **AI & Engine** | `Python`, `LLM`, `Yahoo Finance` | 투자 알고리즘 및 LLM 기반 지표 해설 |
| **Database** | `PostgreSQL`, `Redis` | 마이데이터 저장 및 고속 데이터 처리 |

---

## 🎯 Service Flow (서비스 흐름)
1. **[분석]** 마이데이터 연결을 통한 실시간 지출 패턴 파악
2. **[설계]** 투자 목표 설정 및 AI 기반 가용 자산 확인
3. **[예측]** 시뮬레이션을 통한 목표 달성 가능성 확인
4. **[실행]** AI가 추천하는 종목 및 리밸런싱 가이드로 투자 시작

---

## 👥 사용자 역할
* **일반 사용자(USER):** 내 투자 여력을 알고 싶고, 우량주를 직접 고르고 싶은 초보 투자자
* **관리자(ADMIN):** 재무 데이터 업데이트, 섹터별 가중치 관리 및 시스템 모니터링
